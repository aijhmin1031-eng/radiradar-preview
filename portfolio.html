과기정통부·NTIS·중기부의 방사선·원자력 및 기업지원 공고를 자동으로 수집·선별하고, 새 공고가 뜨면 알려주는 방사선 기업 전용 데스크톱 레이더.

핵심 기능 보기기술·보안
Why

공고는 많은데, 우리 것만 골라내기 어렵다

방사선 기업이 참여할 R&D·기업지원 공고는 여러 부처·기관에 흩어져 있고, 매일 들여다보기 어렵습니다. 놓치면 1년을 기다려야 하죠.

기존 방식

  • 부처·기관별 사이트를 일일이 확인
  • 방사선 관련만 골라내기 번거로움
  • 마감 지난 공고에 시간 낭비
  • 새 공고를 놓침

Features

핵심 기능

수집부터 선별·분석·관리·알림까지, 공고 한 건을 끝까지 따라갑니다.

📡

자동 통합 수집

과기정통부 사업공고(API)·NTIS 국가R&D·중기부 사업공고를 한 번에 수집·중복정리.

🎯

방사선 관련도 선별

공급·제조 / 이용·응용 / 기반·안전 3계층 키워드 + 가중치로 관련 공고만. 원자력 포함.

⏱️

진행중 / 마감 구분

공고문에서 마감일을 추출해 진행중·마감을 배지로 표시. '진행중만' 필터·마감임박 정렬.

🏢

기업지원 트랙

기술실증·고도화·이전·R&D 등 분야무관 지원사업까지 — 방사선 기업도 참여 가능한 공고를 별도 분류.

🔍

규칙기반 자동분석

사업규모·마감·신청자격·요약·신청 체크리스트를 규칙으로 추출(AI 비용 0).

🔖

관심·메모·기록

관심/보관, 메모, 진행상태 기록으로 나중에 이어서 진행. CSV로 로컬 PC에 내보내기.

🔔

자동수집 + 알림

작업 스케줄러로 주기 자동수집, 새 공고가 생기면 Windows 토스트 알림(무설정).

🛠️

키워드 직접 조정

기본 키워드 on/off, STRONG↔MEDIUM 드래그 이동, 내 키워드 추가 — 수집 범위를 직접.

🔐

안전한 설치형

서비스키는 Windows DPAPI로 암호화 저장. 데이터는 내 PC에만. 오프라인에서도 동작.

Two Tracks

"방사선 키워드"에만 갇히지 않습니다. 방사선 기업도 '기업'이니까요.

Engineering

기술 · 보안

가볍고, 안전하고, 비용 없이 도는 데스크톱 제품.

🖥️

데스크톱 네이티브

pywebview(WebView2) + Python. 무거운 런타임 없이 빠르게.

⚙️

규칙기반 · AI 비용 0

키워드·정규식 규칙으로 분류·분석. 호출 비용·외부 의존 없음.

🔐

DPAPI 비밀 보호

서비스키를 이 PC·이 계정에 묶어 암호화. 평문 노출 없음.

📦

설치형 배포

설치관리자로 더블클릭 설치. 데이터·설정은 %LOCALAPPDATA%.

🌐

오프라인 견고성

폰트 번들, 망 차단 환경에서도 디자인·동작 유지.

♿

접근성·절전

키보드 내비·포커스 표시, 비가시 시 애니 정지로 배터리 절약.

Design
